名称

false - （失败地）什么都不做

概述

false [忽略命令行参数]
false 选项

描述

以表示失败的状态值退出。

--help
显示此帮助信息并退出
--version
显示版本信息并退出

注意：您的 shell 可能内置了自己的 false 程序版本，它会覆盖这里所提及的相应版本。请查阅您的 shell 文档获知它所支持的选项。

作者

由 Jim Meyering 编写。
