名称

whoami - 显示有效用户 ID

概述

whoami [选项]...

描述

显示与当前的有效用户 ID 相关联的用户名。等价于 id -un。

--help
显示此帮助信息并退出
--version
显示版本信息并退出

作者

由 Richard Mlynarik 编写。
